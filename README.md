# Folder-Watchdog

This is a personal project and is currently under development.
The purpose of this repository is to integrate several of the newest technologies and methodologies, in order to replicate the full life cycle of a software product.

## <i>Key elements that are desired to be integrated</i>
- [ ] Possibility of monitoring multiple folders simultaneously using parallelism and Observer design pattern
- [ ] Introduce an interactive UI layer for this application using a popular front end technology, such as React
- [ ] Introduce Jenkins pipelines job that automatically validate code integrity, with each pushed commit
- [ ] Run automatic code coverage analyzer and publish the results on the README.md page
